## 3. 공모전 주요 내용

### 참여 방법 (3가지 트랙)

#### 🏆 Track 1 — 인기 작가 도전 (경쟁형)
> "인기 작가 달성해서 보상 받아가세요"

- 앱 내 AI 채팅 캐릭터/시나리오 작품 등록
- 등록 기간 내 누적 좋아요 / 채팅 수 기준 인기 순위 산정
- 상위 랭킹 작가에게 보상 지급

#### ⭐ Track 2 — 작품 등록 + 리뷰 (더블 보상)
> "인기 작품 보상 받고, 작품 등록 리뷰 작성하고 더블 보상 받아가세요"

- 작품 등록 후 SNS(인스타그램 / 틱톡 / X)에 리뷰 콘텐츠 게시
- 지정 해시태그 포함 필수
- 작품 등록 보상 + 리뷰 작성 보상 별도 지급

#### 🎁 Track 3 — 참여 보상 (참가형)
> "작품 등록만 해도 보상을 드려요"

- 기간 내 작품 1편 이상 등록 시 전원 보상
- 진입 장벽 낮춰 신규 작가 대거 참여 유도

## 5. SNS 홍보 전략

### 채널별 역할 분담

| 채널 | 타겟 | 콘텐츠 형식 | 목표 |
|------|------|-----------|------|
| **인스타그램 릴스** | 20~30대 여성 핵심 | 15~30초 숏폼 / 카드뉴스 | 인지도 확산 |
| **틱톡** | 20대 여성 | 챌린지 / 트렌드 활용 숏폼 | 바이럴 확산 |
| **카카오톡 오픈채팅** | 작가 커뮤니티 | 공모전 공지 / 소통 | 작가 참여 유도 |
| **X (트위터)** | 창작자 커뮤니티 | 공모전 공고 / RT 이벤트 | 창작자 네트워크 확산 |
| **네이버 블로그 / 카페** | 웹소설 팬덤 | 공모전 후기 / 정보글 | 검색 노출 |

### 콘텐츠 아이디어 (20~30대 여성 타겟)

**인스타그램 / 틱톡 숏폼**
- "AI가 내 이상형 캐릭터를 만들어준다면?" → 앱 기능 체험 영상
- "작가가 되어 AI 캐릭터를 만드는 과정" → 제작 과정 비하인드
- "공모전 1위 작가의 작품 반응 보기" → 리얼 반응 영상
- "나만의 AI 남친/여사친 캐릭터 만들기 챌린지" → 참여형 챌린지

**해시태그 전략**
```
#AI채팅앱 #AI캐릭터 #공모전 #작가모집 #웹소설작가
#스토익엔터테인먼트 #AI남친 #창작공모전 #신인작가 #캐릭터제작
```

**인플루언서 협업**
- 팔로워 1만~10만 마이크로 인플루언서 3~5명 섭외
- 웹소설 / 창작 / 덕질 콘텐츠 계정 우선
- 공모전 참여 + 리뷰 콘텐츠 게시 형식으로 협업

- **프로세스 검증**: 실제 작가 100명 이상의 창작 과정을 통해 툴 개선점 파악
