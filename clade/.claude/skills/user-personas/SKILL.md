# 유저 페르소나 작성 스킬

## 사용 방법
"페르소나 만들어줘" 또는 "/user-personas [설명]" 으로 호출

---

## 실행 순서

### 1단계 — 컨텍스트 수집 (먼저 질문)
- 몇 개의 페르소나가 필요한가?
- 기존 유저 데이터나 인터뷰 자료가 있는가?
- 집중할 행동/목표가 있는가?

### 2단계 — 페르소나 구조

```
## 페르소나: [이름]

### 기본 정보
- 나이: 20~30대 여성
- 직업:
- 거주지:
- 디지털 활용도: (낮음/중간/높음)

### 목표
- 주요 목표:
- 부수적 목표:

### 불편함 (Pain Points)
- [구체적인 문제 1]
- [구체적인 문제 2]
- [구체적인 문제 3]

### 행동 패턴
- 앱 사용 시간대:
- 주요 사용 기기:
- 비슷하게 사용하는 앱:

### AI 채팅 앱 사용 시나리오
- 언제 쓰는가:
- 무엇을 원하는가:
- 성공 기준 (만족하는 순간):

### 인용 (가상)
> "..."
```

### 3단계 — 품질 기준
- 인구통계가 아닌 행동과 목표 중심으로 정의
- 각 페르소나는 실제 유저 패턴을 반영
- 가정 기반 페르소나는 `[가정: 검증 필요]` 표시
- 스토익엔터테인먼트 AI 채팅 앱 컨텍스트에 맞게 작성
