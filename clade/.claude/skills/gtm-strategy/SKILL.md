# GTM (Go-to-Market) 전략 스킬

## 사용 방법
"런칭 전략 세워줘" 또는 "/gtm-strategy [제품/기능명]" 으로 호출

---

## 실행 순서

### 1단계 — 컨텍스트 수집 (먼저 질문)
- 신규 출시인가, 기능 업데이트인가?
- 타겟 세그먼트 중 가장 확신하는 그룹은?
- 현재 보유한 마케팅 채널은?
- 런칭 예정일은?

### 2단계 — 5단계 GTM 프레임워크

#### Step 1. 시장 조사
- 타겟: 20~30대 여성 세부 세그먼트
- 경쟁사 현황 및 공백
- 고객 검증 데이터

#### Step 2. 마케팅 채널 선정
| 채널 | 적합도 | 이유 |
|------|--------|------|
| 인스타그램/틱톡 | | |
| 유튜브 | | |
| 커뮤니티 (블라인드, 네이트판 등) | | |
| 앱스토어 ASO | | |
| 인플루언서 | | |

> 원칙: 여러 채널을 얕게보다 **소수 채널을 깊게** 공략

#### Step 3. 메시지 전략
- 핵심 가치 제안 (1문장):
- 차별점:
- 타겟 페인포인트 매핑:
- 채널별 메시지 변형:

#### Step 4. 성공 지표 (KPI)
| 단계 | 지표 | 목표값 |
|------|------|--------|
| 인지 | 앱스토어 노출 | |
| 참여 | 다운로드 수 | |
| 전환 | 가입 완료율 | |
| 수익 | 구독 전환율 | |

#### Step 5. 런칭 타임라인
```
[D-30] 사전 준비
- [ ] 앱스토어 등록 자료 준비
- [ ] SNS 채널 세팅
- [ ] 인플루언서 컨택

[D-14] 소프트 런칭
- [ ] 베타 유저 모집
- [ ] 피드백 수집

[D-Day] 정식 런칭
- [ ] 앱스토어 출시
- [ ] 런칭 콘텐츠 발행

[D+30] 모멘텀 유지
- [ ] 지표 리뷰
- [ ] 채널별 성과 분석
- [ ] 다음 액션 결정
```

### 3단계 — 산출물
- 추천 채널 및 이유
- 메시지 전략
- 런칭 타임라인
- KPI 목표
- 리스크 및 대응 방안
- 90일 실행 로드맵
