# 1회차 — 가치 전달 화법

**목적:** "교재 더 푸는 것" → "학습 효과가 달라지는 것"으로 인식 전환

## 비용 저항 극복 화법

- "4만원이 부담돼요"
  → **"주 1회 기준 회당 1만원, 학원 보충수업보다 낮으면서 1:1 맞춤 지도"**
- "교재만 더 푸는 거 아닌가요?"
  → **"OO의 약한 부분만 집중 보강하는 구조. 지난달 진단에서 OO 부분 부족했는데 이걸 잡는 과정"**
- "효과를 모르겠어요"
  → **"3개월 전 시작한 OO 회원은 진단평가 OO점 → OO점. 데이터로 보여드릴게요"**

# 1회차 — 권유 타이밍 가이드

## 황금 타이밍 3가지

- **진단평가 후** → "이 부분을 잡으려면 마.스.타가 효과적입니다"
- **학부모 상담 중 고민 언급 시** → 문제 해결책으로 자연스럽게 연결
- **스마트 구몬 데이터 리뷰 시** → 약점 영역 기반 제안

## 실습
- 교사 2인 1조 롤플레이 (학부모역 / 교사역)
- 3가지 거절 상황별 대응 연습

# 2회차 — 이탈 방어

**목적:** 월 이탈 13명 → 6명 이하로 감소

## 이탈 신호별 교사 조치

- 숙제 미완료 2회 연속 → **분량 30% 축소 → 성공 경험 후 복원**
- "어려워요" 반복 → **1단계 하향, 기초 보강 후 복귀**
- "그만둘까 해요" → **48시간 내 전화 상담, 조정 옵션 제시**

---

# 2회차 — 시간 조율 해결 방안

## 개선안

1. 주 중 **2개 타임슬롯** 제공 (회원이 선택)
2. **비대면 보충 지도** 병행 (스마트 구몬 연계)
3. 교사 간 **회원 시간대 교환** 조율

## 주간 체크리스트
- 마.스.타 회원 전원 금주 학습 완료 여부 확인
- 미완료 회원 개별 연락
- 학습 난이도 불만 회원 진도 조정
- 다음 주 스케줄 사전 확인
