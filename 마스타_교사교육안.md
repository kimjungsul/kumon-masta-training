## 2. 교육 구성 (총 3회차)

### [1회차] 가치 전달 화법 교육 — 4월 첫째 주

**목적:** "교재 더 푸는 것" → "학습 효과가 달라지는 것"으로 학부모 인식 전환

#### 핵심 교육 내용

**A. 비용 저항 극복 화법**

| 학부모 반응 | 기존 대응 (X) | 전환 화법 (O) |
|------------|--------------|--------------|
| "4만원이 부담돼요" | "그래도 효과가 좋아요" | "한 달 4만원이면 주 1회 기준 회당 1만원입니다. 학원 보충수업 1회 비용보다 낮으면서 1:1 맞춤 지도가 됩니다" |
| "교재만 더 푸는 거 아닌가요?" | "아닙니다, 다릅니다" | "마.스.타는 단순 반복이 아니라 OO(아이이름)가 약한 부분만 집중 보강하는 구조입니다. 지난달 진단 결과 OO 부분이 부족했는데, 이걸 잡아주는 과정이에요" |
| "효과를 모르겠어요" | "좀 더 해보세요" | "3개월 전 마.스.타 시작한 OO 회원은 진단평가에서 OO점 → OO점으로 올랐습니다. 데이터로 보여드릴게요" |

**B. 권유 타이밍 가이드**
- 진단평가 후 결과 공유 시 → "이 부분을 잡으려면 마.스.타가 효과적입니다"
- 학부모 상담 시 학습 고민 언급할 때 → 문제 해결책으로 자연스럽게 연결
- 스마트 구몬 학습 데이터 리뷰 시 → 약점 영역 데이터 기반 제안

**C. 실습**
- 교사 2인 1조 롤플레이 (학부모역 / 교사역)
- 3가지 거절 상황별 대응 연습
- 관리자 피드백

### [2회차] 이탈 방어 교육 — 4월 셋째 주

**목적:** 기존 마.스.타 회원 이탈률을 월 13명 → 6명 이하로 감소

#### 핵심 교육 내용

**A. 학습 난이도 관리법**

| 이탈 신호 | 교사 조치 |
|----------|----------|
| 숙제 미완료 2회 연속 | 즉시 학습량 조절 상담 (분량 30% 축소 → 성공 경험 후 복원) |
| "어려워요" 반복 | 현재 단계 1단계 하향 조정, 기초 보강 후 복귀 |
| 학부모 "그만둘까 해요" | 48시간 내 전화 상담, 중단이 아닌 조정 옵션 제시 |

**B. 이탈 예방 체크리스트 (교사 주간 점검)**
- [ ] 마.스.타 회원 전원 금주 학습 완료 여부 확인
- [ ] 미완료 회원 개별 연락 완료
- [ ] 학습 난이도 불만 회원 진도 조정 완료
- [ ] 다음 주 스케줄 사전 확인 완료

**C. 시간 조율 해결 방안**
- 기존: 교사-회원 1:1 시간 고정 → 변경 어려움
- 개선안 1: 주 중 2개 타임슬롯 제공 (회원이 선택)
- 개선안 2: 비대면 보충 지도 병행 (스마트 구몬 연계)
- 개선안 3: 교사 간 회원 시간대 교환 조율

**D. 실습**
- 이탈 위기 회원 시나리오 3건 → 교사별 대응 계획 작성
- 관리자와 1:1 리뷰
